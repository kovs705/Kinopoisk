# Kinopoisk
Search TV shows, movies or cartoons using unofficial Kinopoisk API. 

## Stack
- MVP
- Programmatic UI
- Unofficial [Kinopoisk API](https://kinopoiskapiunofficial.tech)
- SnapKit

## Requirements
- Internet connection
- iPhone or Simulator
- iOS 16.2
